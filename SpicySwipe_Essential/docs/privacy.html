1. Information We Access

SpicySwipe accesses Tinder profile information (such as name, age, bio, interests) and chat history directly from pages you visit on tinder.com. This data is used solely to provide the extension's core features including smart messaging, analytics, and profile analysis.

2. Data Storage

All collected data is stored locally on your device using browser storage. API keys and sensitive settings are encrypted before being stored and are never transmitted elsewhere.

3. Data Transmission

SpicySwipe does not transmit personal data to the developer or any third party. If you choose to enable external AI services (e.g., OpenAI, Gemini, Claude, DeepSeek), limited relevant data (like profile text or chat snippets) may be sent to their APIs strictly for the purpose of message generation.

SpicySwipe does not sell, share, or use any data for advertising purposes.

4. User Control

You can delete your data anytime by removing the extension or clearing your browser storage. You have full control over which AI providers are used and whether features requiring external data processing are enabled.

5. Security

All processing happens locally except for explicitly enabled API interactions. API keys are encrypted on-device and are never shared with the developer or third parties.
